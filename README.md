# Cookies Simple

This is only for testing...

Cookies and Local Storage controller for general use.

## Installation

In a browser not minify:
```html
<script src="cookies.js"></script>
```

In a browser minify:
```html
<script src="cookies-min.js"></script>
```

Using Npm
```shell
$ npm i -g npm
$ npm i --save cookies
```
